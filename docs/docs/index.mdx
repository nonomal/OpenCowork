# 欢迎使用 OpenCowork

OpenCowork 是一个基于 **Electron 36 + React 19 + TypeScript strict** 构建的桌面 AI Agent 工作台。它把聊天、规划、子代理、团队协作、MCP、Cron、消息平台接入、文件预览、SSH 和本地数据库统一放在一个本地优先的应用里。

OpenCowork is a local-first desktop AI agent workbench. It keeps the execution runtime on your machine and uses a narrow preload bridge plus SQLite persistence to coordinate tools, agents, and integrations.

## 当前实现快照 / What ships today

| 领域 | 当前实现 |
| --- | --- |
| 运行时 | 4 层架构：Main / Preload / Renderer / Shared |
| 模型适配 | 5 个运行时 API adapter：Anthropic、OpenAI Chat、OpenAI Responses、OpenAI Images、Gemini |
| 预置提供商 | 24 个内置 provider preset，覆盖 OpenAI-compatible、Anthropic、Google、Ollama、Qwen、Moonshot、MiniMax、Baidu、SiliconFlow、Azure OpenAI、Copilot、Codex 等 |
| 工具系统 | 文件、搜索、Shell、计划、任务、Cron、Goal、Web、Browser、Desktop、MCP、Team、Notify、AskUserQuestion、Widget |
| 消息平台 | 8 个渠道：飞书、钉钉、企业微信、QQ、微信、Telegram、Discord、WhatsApp |
| 应用插件 | 3 个内置 app plugin：Image、Browser、Desktop Control |
| 子代理 | 15 个内置 agent 定义，保存在 `resources/agents/`，首次运行后同步到 `~/.open-cowork/agents/` |
| 技能 | 11 个内置 skill，保存在 `resources/skills/`，首次运行后同步到 `~/.agents/skills/` |
| 命令 | 6 个内置 slash command，保存在 `resources/commands/` |
| 数据持久化 | SQLite (`~/.open-cowork/data.db`) + JSON 配置文件 + 文件系统目录 |

## 系统分层 / Layers

| 层 | 作用 | 关键目录 |
| --- | --- | --- |
| Main | 系统能力、IPC、数据库、MCP、Cron、SSH、消息平台、更新、崩溃日志 | `src/main/` |
| Preload | 安全桥接，只暴露最小必要 API | `src/preload/` |
| Renderer | React UI、Agent Loop、Tool Registry、Stores、Preview、Team Runtime | `src/renderer/src/` |
| Shared | 事件协议、Team Runtime 类型、消息/模型/工具契约 | `src/shared/` |

## 适合什么场景 / Why it exists

- 你想在本地安全地跑 AI Agent，而不是把所有上下文都交给云端。
- 你希望 Agent 不只是聊天，而是能读写文件、跑命令、调用 MCP、发消息、创建计划、排定 Cron。
- 你需要多人协作式的多 Agent 编排，而不是单一“回答器”。
- 你希望把 OpenCowork 接到飞书、钉钉、企业微信、Telegram、Discord、WhatsApp、QQ 或微信里。
